# 7. 其他进制的数字


## 7.1 其他进制表示

在JS中:
* 如果需要表示16进制数字，需要以 0x 开头
  
```js
var a = 0x123;
console.log(a);
```
* 如果需要表示8进制数字，需要以 0 开头

```js
var b = 070;
console.log(b);
```

* 如果需要表示2进制数字，需要以 0b 开头，但是不是所有浏览器都支持

```js
var c = 0b0111;
console.log(c);
```


## 7.2 统一进制表示

像 "070" 这种字符串，有些浏览器会当成8进制解析，有些会当成10进制解析:

```js
a = "070";
a = parseInt(a);
console.log(typeof a, a);
```

解决办法:

在 parseInt() 中传递第2个参数，来指定数字的进制。

```js
a = parseInt(a, 10);
```
