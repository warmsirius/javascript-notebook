# 4. for循环

## 4.1 概念
for语句，也是1个循环语句，也称为 for循环。

在for循环中，为我们提供了专门的位置用来放3个表达式:
* 1.初始化表达式
* 2.条件表达式
* 3.更新表达式


## 4.2 语法

```js
for(初始化表达式;条件表达式;更新表达式){
    语句...
}
```

### for循环执行流程
* 1.执行初始化表达式，初始化变量(初始化表达式之执行1次)
* 2.执行条件表达式，判断是否执行循环
    * 如果为true，则执行循环(3)
    * 如果为false，终止循环
* 4. 执行更新表达式，更新表达式执行完毕，继续重复2
    

for循环的3个部分都可以省略，也可以写在外部。

```js
var i = 0;
for(;i<10;){
    alert(i++);
}
```

```js
// 死循环
for(;;){
    alert("hello");
}
```
